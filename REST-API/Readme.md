# Deno REST API

> This is a REST API using Deno, Oak 

## Run

> Make sur you have installed Deno into your system

```
deno run --allow-net server.ts
```

## Routes

```
GET      /api/v1/products
GET      /api/v1/products/:id
POST     /api/v1/products
PUT      /api/v1/products/:id
DELETE   /api/v1/products/:id
```
